***경로 설정***

절대 경로

1. 최상위 디렉토리부터 순차적으로 경로 작성

2. 드라이브명(C:,D:....)이나 프로토콜명(http://, ftp://...)로 시작

3. 기준 파일의 위치와 상관없이 대상 파일의 경로가 동일

4. 다른 사이트의 자원 사용 시

EX <a href="http://www.naver.com">네이버</a>
<img src="c:/img/main.jpg" alt="다음 로고">
네이버

상대 경로

1) 동일한 디렉토리에 있는 경우

-파일명.확장자

EX <a href="notice.html">공지사항</a>
<img src="logo.jpg" alt="미정국수">
공지사항

2) 하위 디렉토리에 있는 경우

-폴더명 / 파일명.확장자

EX <a href="html/notice.html">공지사항</a>
<img src="images/logo.jpg" alt="puzzle">
공지사항

3) 상위 디렉토리에 있는 경우

-/ : 최상위 디렉토리로 이동

-../ : 한 수준 위로

-../파일명.확장자

EX <a href="html/notice.html">공지사항</a>
<img src="../../3.jpg" alt="doctor">
공지사항

4) 복합 사용

-../폴더명/파일명.확장자
